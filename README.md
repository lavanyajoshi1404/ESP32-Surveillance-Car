# 🚗 ESP32 Surveillance Car

An IoT-based surveillance robot built using **ESP32-CAM** that provides real-time video streaming and remote vehicle control through a responsive web dashboard.

---

## 📌 Features

* 📹 Live video streaming using ESP32-CAM
* 🎮 Real-time car control through a web dashboard
* 📱 Mobile-friendly responsive interface
* 🌐 Wi-Fi based communication
* ⚡ Progressive Web App (PWA) support
* 🔄 Service Worker for improved web experience

---

## 🛠️ Tech Stack

### Hardware

* ESP32-CAM
* DC Motors
* Motor Driver (L298N/L293D)
* Chassis
* Battery Pack

### Software

* Arduino IDE
* HTML5
* CSS3
* JavaScript
* ESP32 Camera Library

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/lavanyajoshi1404/ESP32-Surveillance-Car.git
```

### 2. Upload the firmware

* Open the firmware in Arduino IDE.
* Install the required ESP32 board package.
* Configure your Wi-Fi credentials.
* Upload the code to the ESP32-CAM.

### 3. Open the dashboard

Open the web dashboard in your browser or deploy it using Vercel.
